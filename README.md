# 💰 Sistema Financeiro - FinControl

Um sistema financeiro completo e profissional para gerenciar suas finanças pessoais com dashboard interativo, gráficos e relatórios.

![Sistema Financeiro](https://img.shields.io/badge/status-funcionando-brightgreen)
![Versão](https://img.shields.io/badge/versão-1.0.0-blue)
![Licença](https://img.shields.io/badge/licença-MIT-green)

## 📋 Funcionalidades

### Gestão Financeira
- 💰 **Cadastro de Transações**: Receitas e despesas com categorias
- 📊 **Dashboard Interativo**: Gráficos e resumos
- 🏷️ **Categorias Personalizáveis**: Crie e gerencie categorias
- 🎯 **Metas Financeiras**: Defina e acompanhe objetivos
- 📈 **Relatórios**: Análise detalhada dos gastos

### Recursos Avançados
- 🔍 **Filtros**: Por tipo, categoria, data e pesquisa
- 📊 **Gráficos**: Receitas vs Despesas, Categorias, Evolução
- 📤 **Exportação**: CSV (Excel) e PDF
- 💾 **Persistência**: LocalStorage automático
- 🌓 **Modo Escuro/Claro**: Interface adaptável
- 📱 **Responsivo**: Funciona em todos os dispositivos

### Análise Financeira
- 📊 **Dashboard**: Resumo visual completo
- 📈 **Evolução**: Acompanhamento mensal
- 🏆 **Top Categorias**: Maiores gastos
- 📊 **Relatórios**: Análise detalhada
